Wszystkie funkcje programu OpenCode
📋 Komendy TUI (Terminal User Interface)
- /connect - Konfiguracja dostawcy LLM
- /compact lub /summarize - Kompresja sesji
- /details - Przełączanie szczegółów narzędzi
- /editor - Otwarcie zewnętrznego edytora
- /exit lub /quit lub /q - Wyjście
- /export - Eksport rozmowy do Markdown
- /help - Pomoc
- /init - Utworzenie/aktualizacja pliku AGENTS.md
- /models - Lista dostępnych modeli
- /new lub /clear - Nowa sesja
- /redo - Ponowienie cofniętej zmiany
- /sessions lub /resume lub /continue - Lista i przełączanie sesji
- /share - Udostępnienie sesji
- /themes lub /theme - Lista motywów
- /undo - Cofnięcie ostatniej zmiany
- /unshare - Usunięcie udostępnienia sesji
🔧 Narzędzia wbudowane
- bash - Wykonywanie komend powłoki
- edit - Modyfikacja plików (dokładne zamiany)
- write - Tworzenie nowych plików
- read - Odczyt zawartości plików
- grep - Wyszukiwanie zawartości plików (regex)
- glob - Znajdowanie plików wg wzorców
- list - Lista plików i katalogów
- lsp (eksperymentalne) - Interakcja z serwerami LSP
- patch - Aplikowanie łatek do plików
- skill - Ładowanie umiejętności (SKILL.md)
- todowrite - Zarządzanie listami zadań
- todoread - Odczyt list zadań
- webfetch - Pobieranie treści internetowej
- question - Zadawanie pytań użytkownikowi
👥 Agenty
- Build - Domyślny agent z pełnym dostępem do narzędzi
- Plan - Agent do planowania bez dokonywania zmian
- General - Podagent do zadań ogólnych
- Explore - Podagent do eksploracji kodu
🎯 Funkcje użytkowe
- @filename - Referencja do plików (fuzzy search)
- !komenda - Wykonywanie komend powłoki
- Tab - Przełączanie między Build/Plan mode
- Drag & drop - Wstawianie obrazów
- Multi-session - Wiele agentów równolegle
- Share links - Udostępnianie sesji
- LSP enabled - Automatyczne ładowanie LSP
