# xenia monofont
![xenia title](./assets/title.png)

I made this because monofont should not have to be fugly.\
I use it everyday and I always forget its monospace font.

![xenia specimen](./assets/banner.png)

### Features
* **700+ glyphs** with deep support for symbols and math
* **non_ambiguous characters:** `1`, `l`, `I`, `0`, `O`
* **clean geometry:** no "yucko" lowercase a or g
* **modern:** no typewriter revival
* **python generated:** built with a custom procedural engine
